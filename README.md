| ------ | ------- |  -------- |
| Quanittative / numerical continous    | Continous       | Height of students in a class |

| Qualitative / categorically ordered   | Discrete        | good, fair, poor              |
| Date or Time                          | Discrete or Cont| 25 Jan, 20 dec 2021           |
